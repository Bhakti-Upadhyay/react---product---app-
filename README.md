# 🛍️ React Product App

A simple React-based Product App built for learning front-end development.  
This project demonstrates product listing, product details, cart management, routing, and basic state handling.

---

## 🚀 Features

- 📦 Product Listing Page
- 🔍 Product Detail Page
- 🛒 Add to Cart functionality
- 💳 Checkout page (UI)
- 🔄 React Router navigation
- ⚛️ Context API for global state management
- 🌐 API integration using Axios

---

## 🛠️ Tech Stack

- React JS
- React Router DOM
- Context API
- Axios
- HTML, CSS, JavaScript
- Vite (or Create React App if used)
